# RoFormer-Sim
追一科技发布的simbert升级版本，以RoFormer为基础模型，对SimBERT整合优化得到升级版本的RoFormer-Sim。

在simbert的基础上相当于将基础架构换成了RoFormer,在训练细节上也有区别。并且升级版的RoFormer-Sim有更多的训练语料，不再仅仅包含百度知道中的疑问句，还扩展到了一般句式，适用场景更大。
